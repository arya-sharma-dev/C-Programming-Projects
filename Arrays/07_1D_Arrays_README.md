# 1D Arrays

**9 programs** — These programs cover everything about one-dimensional arrays — storing data, accessing elements, reversing, finding max/min, searching, and merging arrays.

---

## Programs

| File | Description |
|------|-------------|
| sum_average_array.c | Calculate sum and average of all elements in a 1D array |
| swap_first_last.c | Swap the first and last element of a 1D array |
| reverse_array.c | Reverse all elements of a 1D array |
| largest_smallest.c | Find the largest and smallest element in an array |
| linear_search.c | Search for a value in array — returns 1 if found, 0 if not (Linear Search) |
| binary_search.c | Search for a value in a sorted ascending array (Binary Search) |
| merge_asc_desc.c | Merge array A (ascending) and array B (descending) into array C (ascending) |
| merge_desc_desc.c | Merge two descending arrays X and Y into array Z (descending) |
| mix_even_odd.c | MIX() function — combine two arrays by separating even and odd elements |

---

## Concepts Used

- 1D array declaration — `int arr[50]`
- Array input and traversal using loops
- Linear Search — O(n) time complexity
- Binary Search — O(log n) time complexity — works on sorted arrays only
- Merging two sorted arrays into one
- Passing arrays to functions

---

## How to Run

```bash
gcc filename.c -o output
./output
```
