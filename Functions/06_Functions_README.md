# Functions

**6 programs** — These programs show how to write and use functions in C. Covers user-defined functions, return types, passing arguments, pointers for pass-by-reference, and menu-driven programs.

---

## Programs

| File | Description |
|------|-------------|
| sum_function.c | Function that takes two integers and returns their sum |
| factorial_function.c | Function that calculates and returns the factorial of a number |
| primes_in_range.c | Function that displays all prime numbers between two given values |
| power_default_arg.c | Power function — if no exponent is given, default is 2 (squares the number) |
| zero_small_by_ref.c | Function zero_small() — sets the smaller of two numbers to 0 using pointers |
| menu_calculator.c | Menu-driven calculator — separate functions for add, subtract, multiply, divide |

---

## Concepts Used

- Function declaration and definition
- Return types — `int`, `float`, `void`
- Passing arguments to functions
- Pointers — `*` and `&` for pass-by-reference
- Menu-driven programs using `switch`
- Breaking a big program into small reusable functions

---

## How to Run

```bash
gcc filename.c -o output
./output
```
