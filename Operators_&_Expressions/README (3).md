# 📁 Basics & I/O

Programs focused on the very foundation of C — printing output, taking user input, and using basic built-in functions.

---

## 📋 Programs

| File | Description |
|------|-------------|
| `hello_world.c` | Print HELLO WORLD on the screen |
| `subject_marks_table.c` | Display a subject-marks table using a single printf |
| `sum_two_numbers.c` | Accept two numbers from user and print their sum |
| `fahrenheit_to_celsius.c` | Convert temperature from Fahrenheit to Celsius |
| `simple_interest.c` | Accept Principal, Rate, Time and calculate simple interest |
| `ascii_value.c` | Accept a character and display its ASCII value |
| `next_character.c` | Accept a character and display the next character in ASCII |
| `char_type_check.c` | Check character type using isalpha, isdigit, etc. |
| `random_number.c` | Display a random number between 10 and 100 |
| `uppercase_letter.c` | Accept a lowercase letter and display it in uppercase |

---

## 🧠 Concepts Covered

- `printf()` and `scanf()`
- Formatting output with `\t` and `\n`
- ASCII values and character arithmetic
- Built-in functions: `toupper()`, `isalpha()`, `isdigit()`
- `rand()` and `srand()` for random numbers

---

## ▶️ How to Run

```bash
gcc filename.c -o output
./output
```
