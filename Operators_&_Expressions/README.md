# Operators & Expressions

**10 programs** — These programs cover all types of operators in C including arithmetic, assignment, and the ternary operator. Also includes real-world problems solved using operator logic.

---

## Programs

| File | Description |
|------|-------------|
| swap_with_temp.c | Swap values of two variables using a third variable |
| area_of_circle.c | Calculate area of a circle given its radius |
| positive_negative_ternary.c | Check if a number is positive or negative using ternary operator |
| even_odd_ternary.c | Check if a number is even or odd using ternary operator |
| swap_without_temp.c | Swap two variables without using a third variable |
| largest_of_three.c | Find the largest of three numbers using ternary operator |
| currency_notes.c | Break an amount into currency notes (500, 100, 50, 20, 10, 5, 1) |
| days_to_years_months.c | Convert a number of days into years, months and remaining days |
| odd_or_even.c | Check whether a given integer is odd or even |
| absolute_value.c | Find the absolute value of a number |

---

## Concepts Used

- Arithmetic operators — `+`, `-`, `*`, `/`, `%`
- Ternary operator — `condition ? true : false`
- Swap logic with and without temp variable
- Modulo operator for breaking values into parts

---

## How to Run

```bash
gcc filename.c -o output
./output
```
