# C Programming Projects

A collection of C programs written while learning programming from scratch.
This repository documents my hands-on journey through core programming concepts —
from basic input/output all the way to structures, matrices, and nested patterns.

---

## About This Repository

Each program in this repo solves a specific problem or demonstrates a key concept in C programming.
The goal is to build strong fundamentals through practical, working code rather than just theory.

## Topics Covered

### 01 — Basic I/O
Programs covering the fundamentals of C: variables, data types, and basic input/output using `scanf` and `printf`.

### 02 — Operators & Expressions
Programs demonstrating arithmetic, relational, logical, and bitwise operators along with expressions and type conversions.

### 03 — If Else IF Ladder
Programs using conditional statements: `if`, `else`, `else if`, and `switch-case` to make decisions in code.

### 04 — Loop Based Programs
Programs using `for`, `while`, and `do-while` loops to repeat tasks and generate number/star patterns.

### 05 — Functions
Programs demonstrating user-defined functions, function parameters, return values, and recursion.

### 06 — Arrays
Programs working with 1D arrays including linear search, binary search.

### 07 — Strings
Programs that manipulate strings: finding length, reversing, comparing, copying, and searching within strings.

### 08 — 2D Arrays & Matrix Operations
Programs performing matrix operations: addition, subtraction, multiplication, transpose, and diagonal sums.

### 09 — Structures
Programs using `struct` to group related data, create records, and work with arrays of structures.

### 10 — Nested Patterns
Programs generating complex patterns using nested loops — pyramids, diamonds, Pascal's triangle, and more.

### 11 — Sorting
bubble sort, selection sort, and insertion sort.

## How to Run

### Prerequisites
- GCC compiler installed — check with `gcc --version`
- A terminal or command prompt

### Steps

```bash
# 1. Clone the repository
git clone https://github.com/arya-sharma-dev/C-Programming-Projects.git

# 2. Navigate into the project
cd C-Programming-Projects

# 3. Enter the folder you want to run
cd 01_Basics_IO        # replace with any folder name

# 4. Compile the program
gcc FileName.c -o FileName

# 5. Run it
./FileName             # Linux / Mac
FileName.exe           # Windows
```

> **Note:** For programs using math functions like `pow()` or `sqrt()`, add the `-lm` flag:
> ```bash
> gcc FileName.c -o FileName -lm
> ```
