# Structures

**4 programs** — These programs use `struct` to group different types of variables under one name, just like a real-world record. Covers complex numbers, student records, and cricket player stats.

---

## Programs

| File | Description |
|------|-------------|
| complex_number.c | Struct for complex number — perform addition, subtraction, multiplication, division |
| student_failed.c | Array of 25 students — find and display students who failed in more than one subject |
| student_struct.c | Student struct with admission number, name, English/Maths/Science marks, and total |
| batsman_struct.c | Batsman struct with code, name, innings, not-outs, runs — calculates batting average |

---

## Concepts Used

- `struct` declaration and definition
- Accessing struct members using `.` operator
- Array of structures — `struct Student s[25]`
- Functions that work on struct data
- Real-world data modeling — students, cricket players, complex numbers
- Arithmetic operations on custom data types

gcc filename.c -o output
