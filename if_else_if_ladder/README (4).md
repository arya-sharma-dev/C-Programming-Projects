# 📁 Operators & Expressions

Programs using arithmetic, assignment, and ternary operators along with real-world math expressions.

---

## 📋 Programs

| File | Description |
|------|-------------|
| `swap_with_temp.c` | Swap values of two variables using a third variable |
| `area_of_circle.c` | Calculate area of a circle |
| `positive_negative_ternary.c` | Check positive or negative using ternary operator |
| `even_odd_ternary.c` | Check even or odd using ternary operator |
| `swap_without_temp.c` | Swap two variables without using a third variable |
| `largest_of_three.c` | Find largest of three numbers using ternary operator |
| `currency_notes.c` | Break an amount into currency notes (500, 100, 50...) |
| `days_to_years_months.c` | Convert total days into years, months and days |
| `odd_or_even.c` | Check whether an integer is odd or even |
| `absolute_value.c` | Find the absolute value of a number |

---

## 🧠 Concepts Covered

- Arithmetic operators (`+`, `-`, `*`, `/`, `%`)
- Ternary operator (`? :`)
- Swap logic with and without temp variable
- Modulo operator for breaking down values

---

## ▶️ How to Run

```bash
gcc filename.c -o output
./output
```
