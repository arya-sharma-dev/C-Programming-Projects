# If / If-else / If Ladders

**10 programs** — These programs use conditional statements to make decisions. Covers simple if, if-else, nested if-else, and if-else ladders applied to real-world problems.

---

## Programs

| File | Description |
|------|-------------|
| expense_discount.c | Calculate total expenses — apply 10% discount if amount exceeds 5000 |
| profit_or_loss.c | Determine profit or loss based on cost price and selling price |
| youngest_of_three.c | Find the youngest person among three by comparing their ages |
| valid_triangle.c | Check if a triangle is valid — sum of all angles must equal 180° |
| leap_year.c | Check whether a given year is a leap year or not |
| gross_salary.c | Calculate gross salary by applying HRA and DA rules on basic salary |
| telephone_bill.c | Calculate monthly telephone bill based on number of calls (slab system) |
| quadratic_roots.c | Find the roots of a quadratic equation ax² + bx + c using discriminant |
| student_division.c | Find the division of a student — First / Second / Third / Fail based on percentage |
| char_classification.c | Classify a character as uppercase, lowercase, digit, or special character |

---

## Concepts Used

- `if`, `else if`, `else`
- Nested conditions
- if-else ladder for multiple conditions
- Real-world rule-based logic — slabs, grades, salary
- Discriminant formula for quadratic roots

---

## How to Run

```bash
gcc filename.c -o output
```
